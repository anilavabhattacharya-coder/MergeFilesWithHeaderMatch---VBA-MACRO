# 📊 Excel VBA Tool: Merge Files with Header Matching

## 🚀 Overview
This VBA automation tool allows you to merge multiple Excel files into a single base file by dynamically matching column headers.

It eliminates manual copy-paste work and ensures data consistency even when column positions differ.

---

## 🛠️ How It Works

1. Run the macro  
2. Select the **Base File**  
3. Select files to merge (one by one)  
4. Macro matches headers automatically  
5. Data is appended to base file  
6. Click **Cancel** to stop  

---

## 📌 Use Cases

- MIS Data Consolidation  
- Multi-branch data merging  
- Sales / Operations reporting  
- Bulk file aggregation
